Overview. Claude Reflect is a full-stack web application that augments generative AI chat with a post-response epistemic analysis layer. A dual-model pipeline separates primary answer generation from structured Reflect output — surfacing assumptions, confidence zones, completeness gaps, and judgment prompts without replacing human decision-making.

Data Pipeline — Dual-Model Architecture

Two-stage asynchronous flow: Stage 1 streams the primary response; Stage 2 performs epistemic decomposition on the completed output. Identical pipeline for free-form and scenario-based interactions.

Stage 1

Primary Response Generation

Endpoint POST /api/chat · Accepts prompt and conversation history · Returns chunked text/plain stream · Powered by llama-3.3-70b-versatile with automatic fallback to 8b model on rate limit.

Stage 2

Epistemic Reflect Analysis

Endpoint POST /api/reflect · Accepts prompt and primary response · Returns structured JSON: severity, reasoning foundations, confidence topology, completeness gaps, judgment prompts.
